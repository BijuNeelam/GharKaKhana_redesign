# Ghar Ka Khana - Project Summary

## Overview
A modern, responsive web application for Ghar Ka Khana, a cloud kitchen delivering authentic home-cooked meals in Navi Mumbai. Built with Next.js 15, TypeScript, and Tailwind CSS, featuring a complete payment gateway integration.

## Key Features

• **Responsive Web Application**: Modern, mobile-first design with seamless navigation across all devices
• **Menu Management System**: Comprehensive meal plan selection with 6 different plans (Veg Normal/Special, Combo Normal/Special, Non-Veg Normal/Special) with weekly and monthly pricing
• **Payment Gateway Integration**: Full Razorpay integration supporting multiple payment methods (Cards, UPI, Net Banking, Wallets) with secure transaction processing
• **Order Management**: Complete checkout flow with customer information collection, delivery address management, and order confirmation
• **Real-time Payment Status**: Payment verification and status tracking with webhook support for automated order updates
• **Modern UI/UX**: Built with Radix UI components, Tailwind CSS, and Lucide icons for a polished, professional interface

## Technology Stack

• **Frontend**: Next.js 15, React 19, TypeScript, Tailwind CSS
• **UI Components**: Radix UI, shadcn/ui components
• **Payment Gateway**: Razorpay API integration
• **Deployment**: Vercel (automated deployments)
• **Version Control**: Git with GitHub integration

## Personal Contributions

• Created reusable payment components (PaymentForm, PaymentStatus, OrderSummary) following React best practices

## Impact

The redesigned platform provides a seamless online ordering experience, enabling customers to browse meal plans, place orders, and make secure payments. The payment integration supports multiple payment methods, increasing conversion rates and customer convenience.
